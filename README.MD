#Number Analyzer CLI Tool
A python command-line utility that reads numbers from a text file,
performs statistical analysis,and generates a formatted report
##Features
-Calculates total
-Counts numbers
-Finds largest values
-Computes average
-Optional Verbose output
-Custom report filename
##Usage
python analyzer.py data.txt
Verbose mode:
python analyzer.py data.txt --verbose
Custom output file:
python analyzer.py data.txt --output report.txt
