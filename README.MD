#Number Analyzer CLI Tool
A python command-line tool that analyzes numbers from a text file and generates a report
##Features
-Calculates total
-Counts numbers
-Finds largest values
-Computes average
-Optional Verbose output
-Custom report filename
##Usage
python analyzer.py data.txt
Verbose mode:
python analyzer.py data.txt --verbose
Custom output file:
python analyzer.py data.txt --output report.txt
